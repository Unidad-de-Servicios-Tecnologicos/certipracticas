# Diagramas C4 — CertiPrácticas

> Nota: CertiPrácticas es una SPA **100% client-side** (sin backend propio). La única dependencia externa de red es el proveedor de IA (Gemini) cuando se usa la generación asistida. **No hay dictado por voz** en la versión actual.

## 4. Decisiones clave (resumen)

- **Sin backend**: simplicidad de despliegue; persistencia local con `localStorage`.
- **Exportación**:
  - **PDF**: `jsPDF` con texto + `addImage` para logos y firma.
  - **DOCX**: `docx` + `file-saver`, contenido programático.
- **Personalización**:
  - `documentSchema` (logos y capas) + import/export JSON.
  - `canvasHtml` (edición visual rica) con opción de restaurar desde formulario.
- **IA**: opcional, controlada por `VITE_AI_PROVIDER` (Gemini activo).
